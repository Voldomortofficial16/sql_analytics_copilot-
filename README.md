# AI-Powered SQL Analytics Copilot

An AI-powered analytics assistant that converts natural language business questions into SQL queries using Llama 3 and executes them on structured datasets using DuckDB.

## Features

- Upload CSV datasets
- Natural Language to SQL
- Automatic SQL execution
- Interactive visualizations
- Business analytics dashboard

## Tech Stack

- Python
- Streamlit
- DuckDB
- Groq API
- Llama 3
- Plotly

## Example Questions

- Top 10 customers by sales
- Which category generated highest profit?
- Top 5 states by revenue
- Average discount by region
- What is total sales?
